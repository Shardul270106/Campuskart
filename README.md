## Overview
CampusKart allows students to buy, sell, and manage products easily. Features include:  
- **User:** Login/Registration, Marketplace, Cart, Sell Items, Profile & Invoices  
- **Admin:** User Management, Orders, Block/Delete Users  

---

## Problem Statement
Campus communities lack a central, organized platform for buying and selling items. CampusKart solves this with a dedicated, easy-to-use system.  

---

## Tools & Technologies
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Java Spring Boot  
- **Database:** MySQL  

---

## Key Features
- User authentication and profile management  
- Buy, sell, and manage items  
- Cart and order management with invoices  
- Admin dashboard for monitoring users and orders
