# CampusKart

**Project Title:** CampusKart ‚Äì Campus E-commerce Platform  
**One-line Summary:** A campus marketplace for buying, selling, and managing items with user and admin dashboards.

---

## Overview
CampusKart allows students to buy, sell, and manage products easily.

**Features include:**
- **User:** Login & Registration, Marketplace, Cart, Sell Items, Profile & Invoices  
- **Admin:** User Management, Orders, Block/Delete Users  

---

## Problem Statement
Campus communities lack a central, organized platform for buying and selling items.  
**CampusKart solves this** with a dedicated, secure, and easy-to-use system.

---

## Tools & Technologies
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Java Spring Boot  
- **Database:** MySQL  

---

## Key Features
- User authentication and profile management (Update Profile supported)  
- Buy, sell, and manage items  
- Cart and order management with invoice generation  
- Admin dashboard for monitoring users  
- Admin can block or delete user profiles
